physbo.misc.gauss_elim module¶

physbo.misc.gauss_elim.gauss_elim(L, t)¶

Calculate alpha using scipy.linalg.solve_triangular. alpha = (L^T L)^-1 t = L^-1 [(L^T)-1 t]

L: (M, M) array_like A triangular matrix t: (M,) or (M, N) array_like

alpha: numpy.ndarray Solution to the system L^T alpha = t. Shape of return matches t.
